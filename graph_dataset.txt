Concept: Graph

Query Examples:
- What is a graph in graph theory?
- Define graph.
- What does graph mean in graph theory?

Content:
A graph is a mathematical structure used to represent relationships between objects. 
A graph consists of a set of vertices (nodes) and a set of edges connecting pairs of vertices. 
It is commonly represented as G = (V, E), where V is the set of vertices and E is the set of edges.

Concept: BFS Steps

Query Examples:
- What are the steps of BFS?
- Explain the procedure of BFS.
- How does BFS work step by step?

Content:
Steps of BFS:
1. Start from the source vertex.
2. Mark it as visited.
3. Add it to a queue.
4. Remove a vertex from the queue.
5. Visit all its unvisited neighbors.
6. Repeat until the queue is empty.

- Explain the procedure of Dijkstra algorithm.

Content:
Steps of Dijkstra:
1. Set distance of source to zero.
2. Set all other distances to infinity.
3. Select the unvisited vertex with the smallest distance.
4. Update distances of neighbors.
5. Mark the vertex as visited.
6. Repeat until all vertices are visited.

---

Type: Procedure
Concept: Bellman-Ford Algorithm Steps

Query Examples:
- What are the steps of Bellman-Ford algorithm?
- Explain the procedure of Bellman-Ford algorithm.
- How does Bellman-Ford work step by step?

Content:
Steps of the Bellman-Ford algorithm:

1. Initialize the distance of the source vertex to 0 and all other vertices to infinity.
2. Relax all edges repeatedly for |V| - 1 iterations.
3. For each edge (u, v), update the distance if a shorter path is found.
4. After relaxation, check all edges again to detect negative weight cycles.
5. If a shorter path is still found, the graph contains a negative cycle.

---

Type: Procedure
Concept: Prim Algorithm Steps

Query Examples:
- What are the steps of Prim's algorithm?
- Explain the procedure of Prim algorithm.
- How does Prim algorithm build a minimum spanning tree?

Content:
Steps of Prim's algorithm:

1. Start with any vertex as the initial vertex.
2. Add the vertex to the growing minimum spanning tree.
3. Select the smallest weight edge that connects a vertex in the tree to a vertex outside the tree.
4. Add the new vertex and edge to the tree.
5. Repeat until all vertices are included in the tree.

---

Type: Procedure
Concept: Kruskal Algorithm Steps

Query Examples:
- What are the steps of Kruskal algorithm?
- Explain Kruskal algorithm step by step.
- How does Kruskal find a minimum spanning tree?

Content:
Steps of Kruskal's algorithm:

1. Sort all edges in the graph in increasing order of weight.
2. Start with an empty forest where each vertex is a separate tree.
3. Select the smallest edge.
4. Add the edge if it does not create a cycle.
5. Merge the sets of the two vertices.
6. Repeat until the minimum spanning tree is formed.
